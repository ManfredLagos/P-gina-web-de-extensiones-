Documentación

Usuarios

Funcionalidad de Agregar Usuario

El formulario incluye los siguientes campos:

  • Nombre de Usuario.
  • Contraseña.
  • Roles (con opciones de Administrador y Usuario)

Funcionalidad de Agregar Usuario

En la pestaña 'Modificar', se ofrece un formulario que permite seleccionar un usuario existente de la lista desplegable y modificar su nombre de usuario, contraseña y roles.

Lista de Usuarios

Debajo del formulario de modificación, se muestra una tabla con los usuarios existentes, indicando su nombre y rol. Cada usuario tiene un botón 'Eliminar' que abre un modal para confirmar la eliminación del usuario seleccionado.

Políticas

  • Políticas de Alta
  • Políticas de Baja

    • Comando ejecutable Se debe colocar únicamente el comando ejecutable, sin la linea 'echo'

    Ejemplo = Nombre: Allow Calling || Comando Ejecutable: Grant-CsTeamsCallingPolicy -Identity email -PolicyName AllowCalling

    Las politicas funcionan con variables para sustituir los correos, extensiones y numeros, las variables se colocan en los comandos ejecutables en el lugar respectivo donde deben sustituirse, las variables son las siguientes:

    email se reemplaza con los correos introducidos
    extension se reemplaza con las extensiones introducidas
    numero se reemplaza con los numeros introducidos

Funcionalidad de Agregar Política de alta y baja

En la pestaña 'Agregar', se despliega un formulario que permite crear nuevas políticas. Los campos principales del formulario son:

  • Nombre de la política.
  • Comando ejecutable.
  • Prioridad de la política.

Funcionalidad de Modificar Política

En la pestaña 'Modificar', se puede seleccionar una política existente para actualizar sus datos. El formulario muestra los siguientes campos:

  • Nombre de la política seleccionada
  • Comando ejecutable
  • Prioridad

Lista de Políticas

Debajo del formulario de modificación, se muestra una tabla con las políticas existentes, donde se presentan:

  • Nombre
  • Descripción
  • Prioridad

Cada política tiene un botón 'Eliminar' que abre un modal de confirmación para eliminar la política seleccionada.

Cuentas

Administra las cuentas y sus políticas asociadas.

Funcionalidad de Agregar Cuentas

En la pestaña 'Agregar Cuenta', se despliega un formulario que permite crear nuevas cuentas. Los campos principales del formulario son:

  • ID de la cuenta
  • Nombre de la cuenta

Funcionalidad de Modificar Cuentas

En la pestaña 'Modificar Cuenta', se puede seleccionar una cuenta existente para actualizar su ID y su nombre. El formulario muestra los siguientes campos:

  • Nuevo ID de la cuenta
  • Nuevo nombre de la cuenta

Funcionalidad de Agregar Políticas a Cuentas

En la pestaña 'Agregar Políticas/Cuentas', se despliega un formulario que permite agregar políticas a una cuenta seleccionada. Los campos principales son:

  • Selección de la cuenta
  • Selección de las políticas asociadas

(En esta sección se puede seleccionar más de una política seleccionando el icono y para quitar una política el icono )

Lista de Cuentas

Debajo de las pestañas, se muestra una tabla con las cuentas existentes, que incluye los siguientes datos:

  • ID de la cuenta
  • Nombre de la cuenta

Cada cuenta tiene dos opciones de acción:

  • 'Eliminar' para borrar la cuenta seleccionada
  • 'Detalles' para ver más información relacionada con la cuenta y las políticas asociadas

Scripts

Genera scripts para dar de alta o baja, o para cuentas específicas.

Generar Scripts

Funcionalidad del Generador de Scripts

La página contiene diferentes pestañas que permiten generar scripts para altas, bajas o ambos. Cada pestaña muestra un formulario diferente dependiendo de la acción deseada. Los datos que el usuario ingresa en los formularios incluyen:

  • Nombre del archivo
  • Extensiones (Alta o Baja)
  • Políticas a aplicar (seleccionadas de un listado)
  • Opcionalmente, el número local si se activa el switch y la política correspondientes

Scripts por Cuenta

Funcionalidad de Selección de Cuenta y Políticas

El usuario puede seleccionar una cuenta del menú desplegable y, según la cuenta seleccionada, las políticas asociadas se muestran.

Funcionalidad del Generador de Scripts

El formulario principal permite al usuario ingresar el nombre del archivo, extensiones y correos para dar de alta y un número local opcional. El script se genera cuando el usuario envía el formulario. El script generado se muestra en la sección 'preview' y puede descargarse como un archivo .ps1.

Clientes Nuevos

Políticas Clientes Nuevos

Funcionalidad de Selección de política para una cuenta nueva

El usuario puede seleccionar una cuenta (E3 ó E5) del menú desplegable. Según la cuenta seleccionada, las políticas asociadas se muestran en la página. Las políticas se pueden actualizar según sea requerido.

Scripts Clientes Nuevos

Funcionalidad de Selección de Cuenta, Tipo de Cuenta y Políticas

El usuario puede seleccionar una cuenta y un tipo de cuenta del menú desplegable (E3 ó E5). Según la cuenta seleccionada, las políticas asociadas se muestran dinámicamente en la página.

Funcionalidad del Generador de Scripts

El formulario principal permite al usuario seleccionar la cuenta que tiene las políticas asignadas, ingresar el nombre del archivo, elegir un tipo de cuenta (E3 o E5), y un número local opcional. El script se genera cuando el usuario envía el formulario. El script generado se muestra en la sección 'preview' y puede descargarse como un archivo .ps1.
